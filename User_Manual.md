# User Manual
## Chat-App-v2

Welcome to Chat-App-v2, your seamless and real-time team communication platform!

### 1. Getting Started

**Registration & Login**
1. Navigate to the application URL in your web browser.
2. If you are a new user, follow the prompts to register. You can choose a username and password or sign in using Google OAuth.
3. Upon logging in, you will be directed to the main chat interface.
4. You may be prompted to pick a Display Name and avatar. 

**Installing as an App (PWA)**
You can install Chat-App-v2 on your desktop or mobile device for quick access. Look for the "Install App" prompt or use the "Install" option in your browser's address bar/menu.

### 2. Interface Overview

- **User Sidebar (Left):** Displays a list of all users, indicating who is currently online (with a green dot) and showing the "Last Seen" time for offline users.
- **Chat Area (Center):** This is where messages are displayed. 
- **Message Input (Bottom):** Use this area to type messages, add emojis, and attach media files.
- **Admin Controls:** If you are an administrator, you will see extra options for user moderation.

### 3. Messaging Features

**Sending a Message**
- Click on the text box at the bottom of the screen.
- Type your message and hit `Enter` or click the send button.
- Click the smiley face icon to open the rich text and emoji picker.

**Sending Media (Images & Stickers)**
- Click the attachment (paperclip or image) icon next to the message input box.
- Select an image file from your device to upload.
- Images can be viewed in a larger format by clicking on them in the chat.

**Replying to a Message (Threading)**
- Hover over the message you want to reply to.
- Click the "Reply" icon.
- Type your message and send. Your message will indicate it is in response to the selected message.

**Editing and Deleting Messages**
- You can edit or delete messages you have sent.
- Hover over your message and select the "Edit" or "Delete" option.
- Note: Edited messages will display an "(edited)" tag.

**Message Reactions**
- Hover over any message and click the reaction icon.
- Choose an emoji to react to the message.

**Read Receipts**
- Messages display indicators when they have been read.
- Click on the read status (e.g., "Read by 3") to see exactly who has read your message and when.

### 4. Private Messaging (Direct Messages)
- To start a private conversation, click on a user's name in the User Sidebar.
- This will open a private chat room between you and that user.
- All messaging features (media, replies, reactions) work exactly the same in private chats.

### 5. Managing Your Profile
- You can update your display name and view your status.
- Your status ("Online" or "Last Seen") is automatically updated based on your activity.

### 6. Admin Controls (Administrators Only)
If you hold an Admin role, you have additional responsibilities to manage the community:
- **Moderation:** You can ban or temporarily mute users who violate guidelines. 
- Access the Admin Controls panel to view the user list and apply moderation actions.
- Banned users will not be able to participate in chats.

### 7. Troubleshooting
- **Messages aren't sending:** Check your internet connection. The app requires a stable connection to maintain real-time capabilities.
- **Cannot upload image:** Ensure your image is in a supported format (PNG, JPG) and under the maximum file size limit.
- **Issues logging in:** If you used Google to sign up, ensure you use the "Continue with Google" option to sign in.

*For further assistance, please contact your system administrator.*
